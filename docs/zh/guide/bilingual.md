# 双语对照

Felys 不允许混合语言编程（字符串内容除外）。在中文环境下，必须全部使用中文标点符号；在英文环境下，同样要求使用英文标点符号。这主要是为了保证代码的可读性，并强制使用者避免频繁切换输入法。简单来说，目标是在英文输入法下编写代码时，中文字符可以直接通过相同的键位输入。对于一些难以美观兼容的符号，将直接用文字替代。整体而言，中文环境下的文字量会略大一些。

::: info
未提及的部分则无需替换，中英符号都一样。
:::

## 符号替换

- `{}` <=> `「」`
- `()` <=> `（）`
- `""` <=> `“”`
- `''` <=> `‘’`
- `|` <=> `｜`
- `,` <=> `，`
- `;` <=> `；`
- `!` <=> `！`
- `_` <=> `—` 

## 保留字替换

- `and` <=> `和`
- `xor` <=> `异或`
- `or` <=> `或`
- `if` <=> `如果`
- `elif` <=> `否如`
- `else` <=> `否则`
- `while` <=> `循环`
- `return` <=> `返回`
- `true` <=> `真`
- `false` <=> `假`
- `none` <=> `无`

## 比较符替换

- `>` <=> `大于`
- `<` <=> `小于`
- `>=` <=> `大于等于`
- `<=` <=> `小于等于`
- `==` <=> `等于`
- `!=` <=> `不等于`

## 对比示例

中文：

```felys
打印（“你好，世界！”）；
```

英文：

```felys
print("hello, world!");
```
